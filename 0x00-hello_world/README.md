# C - Hello, World

This project introduces C programming basics and compilation process.

## Tasks
0. **Preprocessor** - Script to run C file through preprocessor
1. **Compiler** - Script that compiles a C file but does not link (creates .o file)
2. **Assembler** - Script that generates assembly code from C file (creates .s file)
3. **Name** - Script that compiles a C file and creates executable named 'cisfun'
4. **Hello, puts** - C program that prints a specific string using puts
5. **Hello, printf** - C program that prints a specific string using printf
6. **Size** - C program that prints the size of various data types
## Environment
- Ubuntu 20.04 LTS
- gcc with flags: -Wall -Werror -Wextra -pedantic -std=gnu89
- Betty coding style
